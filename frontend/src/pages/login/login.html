Configurazione Host Richiesta

Stai accedendo tramite IP. Per utilizzare tutte le funzionalità, configura il file hosts:

echo "192.168.1.61 trascendence.be trascendence.fe" | sudo tee -a /etc/hosts

Limitazioni senza configurazione:
• Google OAuth potrebbe non funzionare
• Alcune funzionalità potrebbero essere limitate
• Problemi di CORS possibili
